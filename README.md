# Bolos

En este proyecto creamos un minijugeo simulando los bolos de la vida real.
Lo que hice fue crear el script de la bola y añadí colliders para que se reseteara la escena al momento de caerse la bola de la pista.
La bola puede apuntar más a la izquierda o a la derecha usando las flechas del teclado, y el jugador puede saber a dónde apunta usando el director (un cilindro amarillo en medio de la pista), y se dispara al presionar la barra espaciadora.
